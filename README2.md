# RYUDSWIFT // TERMINAL

Terminal-themed personal site with CRT effects, particle field, and boot sequence.

## Quick Start

```bash
npm install
npm run dev
# → http://localhost:5173
```

## Build & Deploy

```bash
npm run build
npm run deploy  # GitHub Pages
```

## Features

- Boot sequence with typing animation
- CRT scanlines + vignette + flicker
- Interactive Canvas 2D particle field
- Live GitHub stats (cached, with fallback)
- Typing effects on scroll
- Expandable blog articles
- Active nav section tracking
- Fully responsive
